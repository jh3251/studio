# **App Name**: BalanceWise

## Core Features:

- Expense Tracking: Log income and expenses with categories, dates, and amounts.
- Balance Calculation: Automatically calculate and display the current balance.
- Reporting and Visualization: Generate charts and reports to visualize spending habits using an AI tool that chooses how to represent data based on its type and scale.
- Category Management: Define and manage custom expense categories.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to inspire trust.
- Background color: Very light blue (#E3F2FD), almost white, for a clean look.
- Accent color: Muted purple (#9575CD) for interactive elements and call-to-actions.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a balance of modernity and approachability.
- Simple, line-based icons for categories and actions.
- Clean, minimalist design with a focus on readability and ease of use.
- Subtle animations for transactions and category changes.
